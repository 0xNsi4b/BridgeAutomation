
# LayerZero

Софт для автоматизации транзакций черз мосты


## Documentation

```bash
git clone https://github.com/

cd 

pip install -r requirements.txt

python main.py
```

Настройки производятся в папке user_settings, внутри присутствует коментарии, что и где писать
